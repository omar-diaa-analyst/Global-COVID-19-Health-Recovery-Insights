Global COVID-19 Health & Recovery Insights
Project Overview

This project analyzes global COVID-19 data using SQL Server and Power BI to uncover trends in confirmed cases, deaths, vaccination progress, and geographic distribution. The objective is to transform raw healthcare data into meaningful business insights through interactive dashboards and optimized SQL queries.

Business Problem

Governments and healthcare organizations require reliable reporting to monitor pandemic progression across different regions. This project provides an interactive analytical dashboard that enables users to track infections, fatalities, vaccination progress, and geographic hotspots.

Dataset

The project combines two datasets:

CovidDeaths
CovidVaccinations

These datasets were merged into a single master fact table for reporting purposes.

Key Insights
Global confirmed cases exceeded 154 million.
More than 3.25 million deaths were reported.
Europe recorded the highest total death count among continents displayed.
Vaccination rollout accelerated significantly during the final months of the analysis period.
Infection waves showed noticeable increases during late 2020 and early 2021.
Geographic visualization highlighted major outbreak concentrations across multiple regions.
